📄 Resume Screening using Machine Learning
An intelligent Resume Screening System that automatically analyzes resumes and predicts the most suitable job category using Natural Language Processing (NLP) and Machine Learning.This project demonstrates how machine learning can assist recruiters by automating the initial resume filtering process, saving time and improving hiring efficiency.

🚀 Features
📂 Upload and process resumes
🧹 Text preprocessing and cleaning
🧠 Resume classification using Machine Learning
🏷 Predicts the most relevant job category
⚡ Fast and scalable screening process
🛠 Tech Stack

Python
Scikit-learn
Pandas
NumPy
NLTK
Jupyter Notebook

⚙️ Workflow
Dataset Preparation
Collected and structured resumes with labeled job categories.
Text Preprocessing
Lowercasing
Removing special characters
Stopword removal
Tokenization
Feature Extraction
TF-IDF Vectorization
Model Training
Machine Learning classifier trained on processed resumes.

Prediction
The model predicts the most suitable job role for a given resume.
